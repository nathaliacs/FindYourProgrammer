Balão de dicas: quando você clicar no botão para achar a pessoa ao qual você está procurando, serão mostrados todos os repositórios da mesma. Dependendo de como essa pessoa o fez, irá aparecer a linguagem mais usada para a construção do mesmo, descrição (se existir uma) e em todos os casos, título do repositório, ao qual você pode usar como um link para ver o trabalho da pessoa mais de perto. Caso ainda tenha alguma dúvida clique aqui para ir até a página de informações.

Encontre seu programador

Encontre o programador desejado pelo nome do perfil do github dele. Dessa forma a página mostrará os projetos do mesmo e a linguagem deles.

Caso o usuário não exista, não aparecerá resultado.

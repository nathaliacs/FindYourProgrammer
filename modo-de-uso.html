Como funciona

FindYourProgrammer é um site de busca que funciona como ferramenta para empresários ou empresas de tecnologia que precissam selecionar programadores capacitados que estejam de acordo com a sua necessidade. A ferramenta funciona da seguinte forma, nosso mecanismo utiliza a API do Github (site mundial de compartilhamento de projetos relacionados com programação) para analizar os projetos existentes da pessoa no mesmo, e ver qual tipo de projetos desenvolvidos ela apresenta. Dessa forma escolhendo o programador com habilidade necessária para o serviço.

Para usar o buscador siga os seguintes passos:

  • Vá a página principal do site, aqui.
  • Escreva o nome do perfil do Github pertencente ao programador que você deseja contratar.
  • Aperte o botão de busca. Em caso do nome estar escrito corretamente, todos os repositórios com os projetos da pessoa escolhida aparecerão.

Por que procurar um programador

Nosso site é um site de nicho o qual visa ajudar o contratante depois do mesmo já ter feito uma entrevista de emprego com canditados, e o mesmo querendo saber mais sobre o seu canditado, busca visualizar projetos já realizados por ele.
